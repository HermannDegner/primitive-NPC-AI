# Archive: Version 1 - Initial SSD System

このディレクトリには、初期版のSSD（構造主観力学）シミュレーションシステムが格納されています。

## 📁 アーカイブ内容

### メインファイル
- **`main.py`** - 初期版エントリーポイント
- **`simulation.py`** - 初期版シミュレーションエンジン

### 特徴
- 基本的なSSD理論実装
- シンプルな村落シミュレーション
- NPCの基本行動パターン
- 基礎的な狩猟・採集システム

## 🚀 進化の経緯

1. **V1 (初期版)** ← **このアーカイブ**
   - 基本SSD理論
   - シンプルな環境システム

2. **V2 (現行版 - main.py)**
   - 4層SSD統合システム
   - 主観的境界形成システム
   - スマート環境システム
   - 季節システム
   - 捕食者狩猟システム

## 📝 使用方法

```bash
# アーカイブ版を実行する場合
cd archive_v1_initial_system
python main.py
```

## 📊 比較

| 機能 | V1 (初期版) | V2 (現行版) |
|------|------------|-------------|
| SSD理論 | 基本実装 | 4層統合システム |
| 境界形成 | なし | 主観的境界システム |
