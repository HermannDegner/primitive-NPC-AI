## 🧠 主観的境界システムの特徴

- **革新的理論**: 物理的縄張りではなく「内・外」の認知的境界
- **経験ベース**: 相互作用の価値により境界を形成
- **集団境界**: 仲間同士で境界を共有
- **適応的**: 環境とNPCの相互作用で動的に変化
