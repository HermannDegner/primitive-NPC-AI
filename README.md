# 構造主観力学 AIシミュレーション：Primitive NPC

**構造主観力学（SSD）理論は、現実にどのような「作用」をもたらすのか？**

このプロジェクトは、[構造主観力学（SSD）](https://github.com/HermannDegner/Structural-Subjectivity-Dynamics)という独自の理論体系を基盤とし、\*\*「生命らしい振る舞いは、単純な原理から創発するのか？」\*\*という根源的な問いを探求するためのAIシミュレーションです。

NPC（ノンプレイヤーキャラクター）の行動を個別にプログラムするのではなく、SSDの基本原理である\*\*「意味圧」「整合」「跳躍」\*\*の力学を実装することで、予測困難かつ文脈に沿った自律的な行動が生まれる様子を観測することを目的としています。

-----

## 🏛️ 理論の実装：SSD in Action

このシミュレーションのNPCは、SSD理論の概念を直接的なパラメータや機能として実装しています。

### 1\. **構造 (Structure) としてのNPC**

各NPCは、`config.py`で定義された性格プリセットによって、固有の**構造**として生成されます。

```python
# from config.py
PIONEER = {"curiosity": 0.9, "sociability": 0.4, "risk_tolerance": 0.8, "empathy": 0.5}
GUARDIAN = {"curiosity": 0.3, "sociability": 0.8, "risk_tolerance": 0.7, "empathy": 0.9}
```

例えば、\*\*「開拓者 (PIONEER)」**は好奇心 (`curiosity`) が高い構造を持つため、未知のものを求める「跳躍」的な行動を取りやすく、**「守護者 (GUARDIAN)」\*\*は社会性 (`sociability`) や共感性 (`empathy`) が高い構造を持つため、コミュニティの安定を維持する「整合」的な行動を優先します。

### 2\. **意味圧 (Meaning Pressure) としての世界**

NPCは、`environment.py`が生成する世界の様々な出来事を**意味圧**として受け取ります。これには内的なものと外的なものが含まれます。

  * **内的意味圧**: 空腹 (`hunger`)、渇き (`thirst`)、疲労 (`fatigue`) といった、生存に不可欠な生理的欲求。
  * **外的意味圧**: 捕食者 (`Predator`) との遭遇、未知の場所の発見、他者との社会的インタラクション。

`npc.py`内の`calculate_life_crisis_pressure`などの関数は、これらの意味圧を定量化し、NPCの次の行動を決定するトリガーとなります。

### 3\. **整合 (Alignment) と整合慣性 (κ)**

NPCは、発生した意味圧を解消し、安定した状態（**整合**）を維持しようと行動します。例えば、空腹になれば食料を探し、疲労すれば安全な場所で休息します。

このプロセスは`ssd_core.py`で管理され、成功した行動は\*\*整合慣性 (κ, kappa)\*\*として`npc.experience`に蓄積されます。これにより、NPCは経験を通じて特定の行動（狩り、探索、看護など）に習熟していきます。

### 4\. **跳躍 (Leap) としてのモード変化**

安定した整合状態が続くと「退屈」という新たな不整合が生まれます。また、既存の行動では解決できないほどの強い意味圧（`E`, 未処理圧）が蓄積すると、NPCは**跳躍**と呼ばれる非連続的な行動変化を起こします。

このシミュレーションにおける最も重要な跳躍が、`ExplorationModeManager`によって管理される\*\*「探索モード」\*\*への移行です。これは、NPCが日常的な生存活動（整合）のサイクルから脱却し、未知のリソースを求めてリスクの高い探索活動（跳躍）を開始する、劇的な行動パターンの変化です。

-----

## 🚀 主な機能

  * **創発的AI**: NPCの行動は固定されたスクリプトではなく、SSDの力学モデルに基づいてリアルタイムに生成されます。
  * **ダイナミックな社会形成**: `social.py`の`Territory`（縄張り）や`HuntGroup`（狩りグループ）システムを通じて、NPC間の協力、対立、コミュニティ形成といった社会構造がボトムアップで出現します。
  * **信頼と記憶**: NPCは`update_trust`関数を通じて、他者との重要なインタラクション（助けられた、裏切られた等）を「情動的熟量」の高い記憶として刻み、長期的な信頼関係を構築します。
  * **エコシステム**: 獲物や捕食者を含む生態系が、NPCの生存に対する持続的な外的意味圧として機能します。

## 🛠️ 実行方法

1.  **シミュレーションの実行**:
    メインのシミュレーションを実行し、最終的な分析結果を表示します。

    ```bash
    python main.py
    ```

2.  **詳細な分析**:
    複数回のシミュレーションを実行し、より統計的な分析を行うためのスクリプトが用意されています。

    ```bash
    # 総合的な分析
    python comprehensive_analysis.py

    # 捕食者関連の分析
    python predator_analysis.py
    ```

## ⚠️ 構造観照 (Theoria)

このプロジェクトは、構造主観力学という「語り」を用いた一つの思考実験です。シミュレーションから得られる結果は、NPCの行動を「善悪」で評価するためではなく、\*\*「なぜその構造が、その意味圧に対し、そのような反応（整合/跳躍）を選択したのか」\*\*を構造的に理解するためにあります。
